## File Naming
We gotta stick with `camelCase` names for files.
Remember that the case of the first letter of it is lower.

eg:
- `navigationPanel`
- `colorRedSubtle`

## CSS Variables Naming
The obvious case for naming css variables is `kebab`.
It is not about food. Kebab here lowers the case of each letters and uses '-' (dash/minus) instead of spaces.

eg:
- `color-red-subtle`
- `font-sans-serif`
